Tor Browser Bundle: Details

Bundle contents

  • Vidalia (svn trunk revision 2346)
  • Tor 0.2.0.19-alpha (with libevent-1.3e, zlib-1.2.3 and openssl-0.9.8g)
  • FirefoxPortable 2.0.0.12 (includes Firefox 2.0.0.12 and Torbutton 1.1.13-alpha)
  • Polipo 1.0.4

Building the bundle

To re-build the bundle, download the source distribution (signature). See README for the directory layout and changelog. Build instructions can be found in build-scripts/INSTALL.
